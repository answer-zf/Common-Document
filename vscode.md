#  快捷键



|   删除   |      Win快捷键       |                 作用                 |
| :------: | :------------------: | :----------------------------------: |
|          | **Ctrl + Backspace** |        删除光标之前的一个单词        |
|          |    Ctrl + delete     |        删除光标之后的一个单词        |
|          |                      |                                      |
| **跳转** |    **Ctrl + Tab**    |     在已经打开的文件之间进行跳转     |
|          |       Ctrl + P       | 在当前的项目工程里，**全局**搜索文件 |
|          |     **Ctrl + G**     |             跳转到指定行             |
|          |                      |                                      |
